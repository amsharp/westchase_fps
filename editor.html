Map Editor — quick guide

Tools: V select · R road · G surface · B building · P prop
Pan: right-drag, middle-drag, or hold Space + drag  ·  Zoom: mouse wheel · F recenter
Select mode: click an object, drag to move, drag the blue round handle to rotate, drag a white corner square to resize.
  [ ] rotate 5° (Shift = 1°) · arrows nudge · Del remove · type exact size in Properties.
Road: pick a class at left, click to drop vertices, Enter or double-click to finish. In select mode you can drag a road's vertices or the whole road.
Surface / Building / Prop: pick a type at left, click to stamp. The yellow ▲ arrow on a building points out its front (storefront) side, so you can see facing as you rotate.
Undo/Redo: Ctrl+Z / Ctrl+Shift+Z · Fit: F
Save Map downloads westchase_map.json — hand that file to Claude to rebuild the game map. Your work also auto-saves to this browser; Load restores a saved file.
